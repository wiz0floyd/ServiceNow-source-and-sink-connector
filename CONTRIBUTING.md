# Contributing

## Prerequisites
- Java 11 (build enforces `--release 11`)
- Maven 3.x (Windows: `wsl mvn` if Maven not on PATH)
- Docker (for E2E smoke test)

## Build

```bash
# Build (skip tests)
wsl mvn package -DskipTests

# Run unit tests
wsl mvn test

# Run tests for a single class
wsl mvn test -Dtest=HermesSinkTaskTest
```

## E2E Smoke Test

```bash
# Requires Docker
docker compose up -d
./scripts/e2e-smoke.ps1
docker compose down
```

## Pull Request Guidelines

- Branch from `main`; target `main`
- PR titles must match: `feat:`, `fix:`, `chore:`, `docs:`, `test:` (enforced by CI)
- All unit tests must pass: `wsl mvn test`
- JaCoCo check must pass: `wsl mvn verify`
- Spec files and plan files (`docs/superpowers/`) must not be merged to main

## Sensitive Data

Never commit real Hermes keystores, passwords, or instance names. Test fixtures use self-signed certs generated by `src/test/resources/generate-test-certs.sh`.
